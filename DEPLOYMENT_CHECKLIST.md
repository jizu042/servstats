## 📝 Технические детали

### Схема БД (правильная)

```sql
-- Таблица серверов
CREATE TABLE servers (
  id SERIAL PRIMARY KEY,
  host TEXT NOT NULL,
  port INTEGER NOT NULL DEFAULT 25565,
  label TEXT,
  is_active BOOLEAN NOT NULL DEFAULT TRUE,
  created_at TIMESTAMPTZ NOT NULL DEFAULT NOW(),
  updated_at TIMESTAMPTZ NOT NULL DEFAULT NOW(),
  UNIQUE(host, port)
);

-- Таблица сэмплов (для статистики)
CREATE TABLE server_samples (
  id BIGSERIAL PRIMARY KEY,
  server_id INTEGER NOT NULL REFERENCES servers(id) ON DELETE CASCADE,
  ts TIMESTAMPTZ NOT NULL DEFAULT NOW(),
  online BOOLEAN NOT NULL,
  players_online INTEGER NOT NULL DEFAULT 0,
  players_max INTEGER NOT NULL DEFAULT 0,
  ping_ms INTEGER,
  source TEXT NOT NULL DEFAULT 'unknown'
);

-- Таблица сообщений чата
CREATE TABLE chat_messages (
  id BIGSERIAL PRIMARY KEY,
  nick TEXT NOT NULL,
  text TEXT NOT NULL,
  ts TIMESTAMPTZ NOT NULL DEFAULT NOW(),
  is_verified BOOLEAN DEFAULT FALSE
);

-- Таблица сессий игроков
CREATE TABLE player_sessions (
  id BIGSERIAL PRIMARY KEY,
  server_id INTEGER NOT NULL REFERENCES servers(id) ON DELETE CASCADE,
  nick TEXT NOT NULL,
  started_at TIMESTAMPTZ NOT NULL,
  ended_at TIMESTAMPTZ,
  created_at TIMESTAMPTZ NOT NULL DEFAULT NOW()
);
```

## 🐛 Если что-то не работает

### CORS 502 ошибки
**Причина**: База данных не сброшена или сервисы не перезапущены  
**Решение**: Выполните Шаг 1 и Шаг 2 из раздела "Что нужно сделать СЕЙЧАС"

### Uptime сбрасывается
**Причина**: Worker ещё не собрал достаточно данных  
**Решение**: Подождите 10-15 минут после сброса БД

### История игроков пустая
**Причина**: Игроки ещё не заходили на сервер после сброса БД  
**Решение**: Подождите, пока игроки зайдут на сервер

### Скины не загружаются
**Причина**: Все источники скинов недоступны или ник некорректный  
**Решение**: Откройте консоль (F12) и посмотрите логи загрузки скинов

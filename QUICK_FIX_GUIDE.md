## 🐛 Если что-то не работает

### 3D скин всё ещё дефолтный
**Причина**: Все источники скинов недоступны или возвращают ошибку  
**Решение**: 
1. Откройте консоль (`F12`)
2. Посмотрите логи загрузки скинов
3. Проверьте, что ник игрока корректный
4. Попробуйте открыть URL скина напрямую в браузере

### Аватар в сайдбаре чёрный
**Причина**: Ошибка загрузки изображения  
**Решение**:
1. Проверьте консоль на ошибки
2. Попробуйте выйти и войти заново
3. Проверьте, что API возвращает правильный URL аватара

### Сообщения дублируются
**Причина**: Redis и прямой broadcast работают одновременно  
**Решение**: Проверьте, что используется только один метод (исправлено в коммите)

### Данные не парсятся
**Причина**: Все API источники недоступны или возвращают ошибки  
**Решение**:
1. Проверьте логи Worker в Render Dashboard
2. Убедитесь, что `MONITOR_SERVERS` настроен правильно
3. Проверьте, что сервер Minecraft доступен
